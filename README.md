# Browser Locker

Visit on [Chrome Web Store](https://chromewebstore.google.com/detail/ioebechakfmaoboaimphhmmjkjjanamn)  

Important Notice ⚠️:
1)  If you have similar extensions installed, please make sure to disable or remove them first. 🚫
2)  Don’t forget your set password. 🔐

This extension doesn’t require any special permissions and can be used with confidence. 👍 
It offers only two features:
1) Password required to start the browser. 🔑
2) Click the extension icon to quickly close all windows and lock the browser (you can also use the shortcut Ctrl+Shift+L). 🖱️
